#### 🔬 FOSSology — Google Summer of Code 2025

I spent a summer building a complete microservices infrastructure for FOSSology — scheduler, database, and agent components running in Docker and Kubernetes. It taught me distributed systems, configuration management, and what happens without proper observability: entire afternoons spent manually correlating logs across containers to figure out why the scheduler was crashing or why builds were slow. That pain is exactly what later drove my OpenTelemetry work.
